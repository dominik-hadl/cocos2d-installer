CCAnimation Class Reference

Inherits from NSObject
Conforms to NSCopying
Declared in CCAnimation.h

Overview

A CCAnimation object is used to perform animations on a CCSprite object. The CCAnimation object primarily contains a collection of CCAnimationFrame objects to define the animation.
